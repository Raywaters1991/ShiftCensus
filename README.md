# ShiftCensus

A scheduling and census management tool for SNFs.

## Branches
- master: stable production-ready code
- dev: active development

## Setup Instructions
1. Clone repo
2. Run backend: npm install && npm run dev
3. Run frontend: npm install && npm start

## Features
- Custom date picker
- Role-based shift logic (RN/LPN/CNA)
- Template builder
- Overtime tracking
- Dark/light mode
